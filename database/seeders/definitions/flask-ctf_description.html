Welcome to the Capture The Flag (CTF) Challenge!

In this scenario, you'll be diving into a simple web application built using Flask. The goal is to find the hidden flag, which is a SHA256 hash starting with CTF .

Each time you launch this challenge, you will encounter a randomized scenario. This means the vulnerabilities and the methods to find the flag will vary, so stay sharp and explore thoroughly!

Key Points to Remember:

  • All vulnerabilities are simple and do not require any extra tools.
  • There are 5 possible scenarios, each hiding a flag on a different place.
