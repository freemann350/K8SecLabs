Welcome to the Brute Force Attack Training Environment!

In this scenario, you'll be exploring and learning about brute force attacks by targeting an SSH service. This environment consists of two main components: an SSH server and a Kali Linux instance with a web-based VNC interface for your attack operations.

Key Features:

  • SSH Server: The SSH server is configured with a user account that has a crackable password. Your objective is to discover this password through brute force techniques.
  • Kali Linux Environment: A Kali Linux container provides a comprehensive set of tools for conducting brute force attacks. Access it via a web-based VNC interface.

Components:

  • SSH Server: This machine runs an SSH service with the one of the values of each wordlist:
    • Usernames
    • Passwords
  • Kali Web: This container runs a Kali Linux instance with a web-based VNC interface. Use this interface to launch your brute force attacks and other penetration testing activities.

How to Start:

Access the Kali Linux environment via the VNC interface via this link. The VNC password will be provided by the lecturer.

Once you have access to the Kali Linux environment, open a terminal and use tools like Hydra or Medusa to start brute forcing the SSH service running. The target SSH is located within the same network.

Common Tools and Techniques:

  • Hydra: A parallelized login cracker which supports numerous protocols.
  • Medusa: A speedy, parallel, and modular login brute-forcer.
  • John the Ripper: Primarily a password cracker, but also supports SSH brute force.
