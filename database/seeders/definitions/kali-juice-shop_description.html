Welcome to the OWASP Juice Shop Training Environment with Kali!

In this scenario, you'll be exploring and learning about common web vulnerabilities through the OWASP Juice Shop application, alongside utilizing a Kali Linux instance with VNC for penetration testing purposes. This dual environment is designed to enhance your understanding of web security in a practical, hands-on manner.

Key Features:

  • Variety of Vulnerabilities: The Juice Shop includes a wide range of vulnerabilities covering most of the OWASP Top Ten.
  • Realistic Application: The environment mimics a real-world e-commerce application, enhancing your practical skills.
  • Educational Resources: The application provides hints and explanations to help you learn as you explore.
  • Kali Linux with VNC: A pre-configured Kali Linux environment accessible via VNC for running additional tools and scripts.

Common Vulnerabilities to Explore:

  • SQL Injection: Try to manipulate database queries through user inputs.
  • Cross-Site Scripting (XSS): Inject malicious scripts into web pages viewed by other users.
  • Cross-Site Request Forgery (CSRF): Perform unauthorized actions on behalf of other users.
  • Insecure Direct Object References (IDOR): Access data or perform actions that should be restricted.
  • Security Misconfigurations: Identify and exploit incorrect configurations and settings.
